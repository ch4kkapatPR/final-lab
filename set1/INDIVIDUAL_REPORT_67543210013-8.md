## ขอบเขตงานที่รับผิดชอบ

## สิ่งที่ได้ดำเนินการด้วยตนเอง
- แก้ Frontend ไม่แสดง Task
- แปลง Password ให้เป็น Hash เพื่อให้สามารถ Login ผ่าน auth service ได้
- แก้ Frontend เรียก API ไม่ได้

## ปัญหาที่พบและวิธีการแก้ไข
Frontend ไม่สามารถเรียกใช้ API ได้ตอนเข้าหน้าเว็ปจึงไม่แสดง Task โดยทำการเพิ่มฟังก์ชัน apiFetch เข้าไปในส่วนของ Scripts 

## สิ่งที่ได้เรียนรู้จากงานนี้
ได้เรียนรู้การ Debug ในส่วนต่างๆของ Frontend เมื่อไม่สามารถเชื่อม Frontend กับตัว Backend ได้
